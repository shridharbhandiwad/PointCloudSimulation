Point Cloud Generation

Realistic Surface Sampling:

  • Points sampled on visible faces (front/back, sides, top)
  • Weighted towards radar-facing surfaces
  • Corner reflector effect simulation (2x RCS at corners)
  • Distance-based detection count scaling

Motion Model

  • Direction: Incoming (towards radar) or Outgoing
  • Speed Control: IDM-like car-following with time headway
  • Loop Mode: Continuous circulation or single pass
  • Acceleration: a_max = 2.0 m/s¬≤, b_max = 4.0 m/s¬≤

Dual Model

1. DBSCAN Clustering

Density-based spatial clustering to group radar detections into objects.

  • Epsilon (Œµ): Maximum neighbor distance (default: 2.0m)
  • Min Samples: Core point threshold (default: 2)

Extracted Features

  • Centroid (x, y)
  • Extent (width √ó length)
  • Point count
  • Mean/Std RCS
  • Mean/Std radial velocity

2. Multi-Object Tracker

Maintains persistent object IDs across frames using nearest-neighbor association.

  • Association: Greedy distance-based matching
  • Motion Model: Linear prediction (constant velocity)
  • Track Lifecycle: min_hits to confirm, max_misses to delete

Track State

  • Position (x, y)
  • Velocity (vx, vy)
  • Age, hits, misses
  • Feature history (last 20 frames)

3. Classification Models

Rule-Based Classifier

Hand-crafted feature thresholds for each class:

  • Point count ranges
  • Extent boundaries
  • RCS level thresholds
  • Speed constraints

Fast, interpretable, no training required

Naive Bayes Classifier

Probabilistic model with Gaussian feature distributions:

  • Pre-trained mean/std per feature per class
  • Prior probabilities
  • Log-likelihood computation
  • Posterior probability output

Probabilistic confidence, adaptable

Usage Guide

Quick Start

# Install dependencies pip install numpy PyQt5 pyqtgraph pyopengl # Run the application python app.py

Typical Workflow

  1. Configure World: Set road dimensions and lane count
  2. Configure Radar: Position, FOV, range, noise parameters
  3. Add Traffic: Select type, direction, speed, lane
  4. Enable Output: Configure UDP/CSV as needed
  5. Start Simulation: Click Start in toolbar
  6. Monitor: Observe 3D view and classification panel
  7. Analyze: Open Bird's Eye View for 2D metrics
